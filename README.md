# GitLab Setup and Basics Guide (CSE)

This repository contains documentation designed to help students
**get started with CSE GitLab** and understand the **fundamental GitLab workflow**
used throughout the course.

The guides are written for beginners and are based on real command-line
and GitLab web interface examples.
